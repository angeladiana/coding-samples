# Coding Samples
This repository contains selected coding samples demonstrating data cleaning and econometric analysis using Stata and R.

## Stata

### 01_FIES_Cleaning.do
**Family Income and Expenditure Survey (FIES) Data Cleaning**
Preparation of household-level survey data, including data cleaning, variable construction, and generation of analysis-ready datasets.

### 02_FIES_Analysis_Heckman.do
**Master’s Final Paper: Who Benefits from Private Pension Programs? An Analysis Across Occupations and Industries**
Estimation of a two-stage Heckman Selection Model using household-survey data.

## R

### 03_ClimateData_Cleaning.R
**Climate Data Processing**
Automated downloading, cleaning, and processing climate data from the Open-Meteo database used in rice price forecasting models.

## Purpose
This repository serves as a portfolio of coding work relevant to research assistantship, predoctoral research positions, and PhD applications.
